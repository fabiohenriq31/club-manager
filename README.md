# Club Manager

Club Manager is a football club management simulator focused on administration instead of coaching.

The player becomes the president of a football club and manages:

- Finances
- Sponsors
- Players
- Board
- Youth Academy
- Infrastructure
- Marketing
- Competitions

## Tech Stack

- React
- TypeScript
- Fastify
- PostgreSQL
- Prisma
- Turborepo
